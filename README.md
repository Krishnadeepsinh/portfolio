# Krishnadeepsinh Chudasama - Portfolio

A highly immersive, futuristic, cinematic 3D portfolio website showcasing the work and skills of Krishnadeepsinh Chudasama, a Computer Science student specializing in Full Stack Development.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion
- **3D Graphics**: Three.js, @react-three/fiber, @react-three/drei
- **Deployment**: Vercel
